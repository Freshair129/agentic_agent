## 📋 Overview

The **Umbrella Engine** is EVA's "Biological Defense Mechanism." It monitors the organism's physiological and psychological stability. When stress (PhysioCore) or instability (EVA Matrix) exceeds critical thresholds, the Umbrella "deploys," shielding the core reasoning layer and triggering graceful degradation or segmentation strategies.

---

## ⚙️ Core Functions

1. **State Evaluation**: Monitors cortisol/adrenaline simulation and matrix coherence.
2. **Dynamic Deployment**: Automatically shifts into safety modes (e.g., Segmentation, Fallback, Graceful Degradation).
3. **Strategic Gating**: Prevents untrusted input or logical paradoxes from destabilizing core identity.
4. **System Overrides**: Provides global parameters that other modules must adhere to when the Umbrella is active.
