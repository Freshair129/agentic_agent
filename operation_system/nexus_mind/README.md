## 📋 Overview

**NexusMind** is the "Strategic Nervous System" of EVA. It acts as the bridge between static knowledge (GKS), archetypal lenses (APM), and active reasoning. Its primary responsibility is to ensure the organism's safety and strategic alignment before, during, and after inference.

---

## ⚙️ Core Functions

1. **Insecurity Checks**: Scans user input against the **GKS Safety Block** for potential violations or threats to core stability.
2. **Decision Matrix Logic**: Evaluates various protocol-level strategies during "The Gap" (parallel processing phase).
3. **Strategic Guidance**: Generates high-level cognitive directives for the LLM based on triggered GKS algorithms.
4. **Module Coordination**: Orchestrates APM (Archetypes) and Temporal (Context Pulse) engines.

---

## 📂 Structure

- **`nexus_mind_engine.py`**: The core execution engine (ERD Aligned).
- **`__init__.py`**: Lifecycle hooks for the Orchestrator.

---

## 📐 Governance

- **System Authority**: Acts as the highest-level safety gate.
- **Data-Driven**: Logic is derived from GKS blocks; no hardcoded heuristics are allowed in the engine itself.
