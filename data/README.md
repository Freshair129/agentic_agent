# EVA Data Assets

**Directory**: `data/`  
**Purpose**: Static data assets and curated knowledge files.  
**Version**: v9.6.2

---

## 📋 Overview

The **Data** directory stores curated, static information used by EVA for personalization, context grounding, and domain-specific knowledge.

---

## 📂 Notable Files

- **`resume_data.json`**: Personal information and professional record used by the user-facing resume web component and for autonomous personality grounding.

---

## 📐 Usage

These files are typically consumed by:

- **GKS (Genesis Knowledge System)** for grounding.
- **RAG Services** for retrieval.
- **Tools** for specific user tasks (e.g., updating a resume).
